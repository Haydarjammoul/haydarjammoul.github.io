HOW TO USE

1. Unzip this archive.
2. Replace photo.jpg with your portrait (same filename).
3. Edit index.html if you want to change text.
4. To preview locally, open index.html in your browser.
5. Upload the files to your GitHub Pages repo (repository named exactly: haydarjammoul.github.io).
   - You can drag-and-drop files in the GitHub web UI or push via git:
     git add .
     git commit -m "Add modern website"
     git push origin main

Site will be live at: https://haydarjammoul.github.io
